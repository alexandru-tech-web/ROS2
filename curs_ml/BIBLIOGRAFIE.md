# BIBLIOGRAFIE.md -- referinte per parte

Principiu: teoria din curs preda complet; referintele sunt pentru APROFUNDARE,
nu inlocuitori. Se prefera versiunile legale gratuite (multe carti sunt oferite
de autori in PDF). Verifica intotdeauna linkul oficial al autorului/editurii.

## Fundamente, regresie, clasificare, arbori, nesupervizat

- Hastie, Tibshirani, Friedman -- The Elements of Statistical Learning (ESL).
  PDF gratuit oferit de Stanford. Referinta principala pentru M03-M16.
- James, Witten, Hastie, Tibshirani -- An Introduction to Statistical Learning
  (ISL), statlearning.com. Varianta mai blanda a ESL, cu laburi.

## Probabilistic / bayesian

- Bishop -- Pattern Recognition and Machine Learning (PRML). PDF gratuit oferit
  de Microsoft Research. Pentru M01, M08, M17.
- Murphy -- Probabilistic Machine Learning (Introduction si Advanced). Drafturi
  oferite de autor.

## Matematica pentru ML

- Deisenroth, Faisal, Ong -- Mathematics for Machine Learning (MML). PDF gratuit
  (mml-book.com). Pentru M00-M02.

## Optimizare

- Boyd, Vandenberghe -- Convex Optimization. PDF gratuit (Stanford). Pentru M02.

## Deep learning

- Goodfellow, Bengio, Courville -- Deep Learning (deeplearningbook.org). Pentru M20.
- Zhang et al. -- Dive into Deep Learning (d2l.ai), cu cod executabil. Pentru M19-M20.

## Invatare prin recompensa (RL)

- Sutton & Barto -- Reinforcement Learning: An Introduction, editia 2. PDF
  gratuit oferit de autori. Pentru M21-M22.

## Practic / cod

- Geron -- Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow.
  Pentru partea de validare cu scikit-learn (fisierele `*_sklearn.py`).

## Articole de cercetare

- arXiv.org pentru lucrarile primare (SHAP, calibrare, conformal prediction,
  gradient boosting etc.). Citate punctual in modulul relevant.
